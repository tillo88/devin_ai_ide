# Archive

This folder keeps historical project material that should not clutter the active root, but may still be useful for rollback, audits, or context recovery.

- `old_docs/`: previous planning/status documents superseded by `README.md`, `README_DEVIN_AI_IDE.md`, `ROADMAP_DEVIN_UI.md`, and `docs/CONTINUITY_2026-07-15.md`.
- `old_tests/`: old generated dumps, diagnostics, checksums, and pre-cleanup `.orig` backups.

Policy: prefer archiving over deleting. If a file becomes useful again, move it back intentionally and document why.
